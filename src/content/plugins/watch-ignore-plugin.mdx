[watch 모드](/configuration/watch)인 경우, 제공된 경로나 정규 표현식에 해당되는 특정정 파일을 무시합니다다.

```javascript
new webpack.WatchIgnorePlugin({ paths });
```

## Options

- `paths` (`Array<string | RegExp>`): 정규 표현식 또는 무시되어야 하는 디렉터리나 파일까지의 경로 목록.
