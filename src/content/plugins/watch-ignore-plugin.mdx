[읽기 모드](/configuration/watch)인 경우, 제공된 경로나 정규 표현식같은 지정된 파일을 무시하세요.

```javascript
new webpack.WatchIgnorePlugin({ paths });
```

## Options

- `paths` (`Array<string | RegExp>`): 정규 표현식이나 절대 경로의 목록에서 디렉터리나 무시되어야 할 파일까지의 경로.
